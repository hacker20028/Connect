
# Connect

Connect is a Chat Application developed using Flutter. This Application aims to provide the most sercured chatting environment.


## Features
---
- Hassle free registration/Login
- Cross platform
- Send chat messages
- Send Images
- Update profile
- last seen
- Time stamp
- Message read tick
- View Other's profile
